Document overlayer

This app is meant to overlay multiple documents in order to get an overview of the differences between these documents. To get started, select multiple files from your dropbox account. You can select multiple files by holding shift.

This app uses functionality provided by Google Docs. Please make sure you are logged into your Google account.
